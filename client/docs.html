Veestream API Documentation

Welcome to the Veestream API documentation. Veestream is a file management API that allows you to upload, progressively download, and manage your files . This API is intended for anyone who wants an efficient file management solution that allows your users to access the files efficiently via REST endpointsts.

Base URL

The base URL for the Veestream API is https://veestream.tech.

Getting an API key

An API key is required to use the veestream API. To get an API key,register by providing your name, email and paying 1$ via paypal, credit card or debit card. After you signup, you will get an API key. It is recommended that you keep this API key safe and private. This API key must be provided in the request header when making calls to other API endpoints. After the payment, you will get an email from paypal with your transaction receipt. It is recommended that you keep this receipt safe for reference.

Endpoints

Upload File

This endpoint allows a user with an API key to upload one file at a time. When you signed up, an azure blob storage container is created for you. All the video files you upload will be stored on that container.

POST /file/upload

Headers

Use the following header in your request for the upload endpoint:

apikey: yourapikey

Body

Include name in the request body. This should be a string, the name for your file .

name:"video name"

File Upload

To upload a file, send a POST request to the API endpoint and include the file in the request. Also add a name in the request body

The following example demonstrates how to upload a file using curl

curl -X POST -H "apiKey: b7642f0d6db02f383498e7f7edd353597ad8e5fe" -F "video=@/home/amschel/Downloads/honey.mp4" -F "name=My Video" https://veestream.tech/file/upload

The following example demonstrates how to upload a file using javascript


	const apiKey = 'b7642f0d6db02f383498e7f7edd353597ad8e5fe';
	const fileInput = document.querySelector('#file-input');
	const formData = new FormData();
	formData.append('video', fileInput.files[0]);
	formData.append('name', 'My Video');
	
	fetch('https://veestream.tech/file/upload', {``
	  method: 'POST',
	  headers: {
		'apiKey': apiKey,
	  },
	  body: formData
	})
	.then(response => response.json())
	.then(data => console.log(data))
	.catch(error => console.error(error));

Get file url for progressive download

GET /file/:id

This endpoint returns a valid file url that can be used to download the file, render the file on the web if it was an image, and as an src attribute of the Html5 video tag to play videos in browsers. 

GET /file/61f19d85d28a103995b1816c

To make a request to this endpoint, pass a valid apikey in the request headers. If the apikey is wrong, you will get a 401 anauthorized response.


				const videoPlayer = document.querySelector('video');
				const apiKey = 'your_api_key_here';
				const videoId = 'video_id_here';
				const apiUrl = `https://veestream.tech/file/${videoId}`;
				
				fetch(apiUrl, { headers: { apikey: apiKey } })
					.then(response => response.json())
					.then(videoUrl => {
						videoPlayer.src = videoUrl;
						videoPlayer.play();
					})
					.catch(error => console.error(error));

Get File Metadata

GET /file/:id/data

This allows you to get metadata about a particular file. We pass the id parameter to specify the file we are requesting data for. id is from _id field that each file has.

Below is an example curl command

curl -X GET -H "apikey: b7642f0d6db02f383498e7f7edd353597ad8e5fe" https://veestream.tech/file/61f19d85d28a103995b1816c/data

Below is the example code in javascript for this endpoint

const axios = require('axios');

const url = 'https://veestream.tech/file/61f19d85d28a103995b1816c/data';

const headers = {
  apiKey: 'b7642f0d6db02f383498e7f7edd353597ad8e5fe',
};

axios.get(url, { headers })
  .then((response) => {
    console.log(response.data);
  })
  .catch((error) => {
    console.log(error.response.data);
  });

Generate a video thumbnail

A video thumbnail is generated by taking a screenshot of the video by default from 0-2 secs.You can also pass the start and end of where you want to take the thumbnail image in request query. It is of type image/png. Below is the example curl command to generate a thumbnail. 


curl -X GET https://veestream.tech/file/61f19d85d28a103995b1816c/poster?start=5&end=7 -H 'apikey: b7642f0d6db02f383498e7f7edd353597ad8e5fe'

When you make a request to /file/:id/data, you get a poster as part of the response. You can add the base url i.e https://veestream.tech and concatenate it with the poster to create the full url for the thumbnail endpoint. You can set this url as the src attribute of an image to display video thumbnails on your frontend.

Check how to generate thumbnail on the 'getVideoMetaData' example on github

Generate Video GIF

A video gif is generated using ffmpeg which takes a video and generates a 2 seconds gif from 10-12 seconds of the video by default. 


		curl -X GET https://veestream.tech/file/61f19d85d28a103995b1816c/gif?start=5&end=10 -H 'apikey: b7642f0d6db02f383498e7f7edd353597ad8e5fe'

When you make a request to /file/:id/data, you get a gif too as part of the response. You can concatenate the base url, https://veestream.tech with the gif to create the full url for the gif endpoint.You can now set the url as an src attribute of an image on hover to play the gif when a user hovers over the image.

Check how to generate gif on 'getVideoMetaData' example on github

Get All Files

GET /files

This endpoint allows you to get all the files related to your account. It returns an array of file metadata for all the files stored in your account. Below are examples of how to utilize this endpoints using curl and javascript

curl:
curl -H "apikey: b7642f0d6db02f383498e7f7edd353597ad8e5fe" https://veestream.tech/files

Javascript


			fetch('https://veestream.tech/files', {
			headers: {
			  'apikey': 'b7642f0d6db02f383498e7f7edd353597ad8e5fe'
			}
		  })

			.catch(error => console.error(error));
